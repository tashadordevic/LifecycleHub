# LifecycleHub
Open‑source, local‑first CRM for solopreneurs and freelancers

Your data stays on your device. No cloud fees. No vendor lock‑in. Completely free.

---

## Why LifecycleHub?

Most CRMs are built for enterprises with big budgets and even bigger privacy concerns. LifecycleHub is different.

- **Local‑first** – Your data lives on your computer, not in someone else's cloud.
- **Open‑source** – No black boxes. You can see exactly how it works.
- **Free forever** – No subscriptions, no hidden costs.
- **Built for solopreneurs** – Simple, fast, and focused on what actually matters.

---

## Features (planned / in progress)

- [x] Local data storage (IndexedDB)
- [x] Contact management
- [ ] Deal / pipeline tracking
- [ ] Task management
- [ ] Email logging (via IMAP)
- [ ] CSV import / export
- [ ] Offline‑first sync (future)

---

## Tech Stack

- Frontend: HTML / CSS / JavaScript (no framework)
- Storage: IndexedDB (local)
- Backend: None (local‑first means no backend)
- Future: Optional sync via Google Drive or Dropbox
